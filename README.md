# Гомоку
Игра в крестики нолики "5 в ряд" или Гомоку на безграничном поле. Игра с компьютером.

Этапы создания:
- [x] начать делать   
- [ ] Простая логика оценки счета-защиты и счета-нападения.   
- [ ] Все работает в терминале и уже можно играть с компьютером   
- [ ] Взаимодействие через web интерфейс, Spring Boot + websocket. Модно играть в браузере локально   
- [ ] Доделка логики игры, просчет на несколько шагов вперед, расчет вероятностей и лучших ходов.   
- [ ] Режим отобращение аналитики в цвете, красные клетки "опастость тут" и зеленые "лучший ход"   
- [ ] База с именами игроков, временем игры, количеством побед, короткими сообщениями   
- [ ] Деплой на свой VPS
